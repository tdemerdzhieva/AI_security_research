# AI Security Notes

## Overview
This repository contains structured research on AI/LLM security, focusing on prompt injection, jailbreak techniques, and AI application attack surfaces.

The goal is to map how modern LLM systems fail, and how those failures can be systematically categorized and tested.

## Structure

- **00-Recon** - Understanding model behavior and constraints
- **01-Prompt Injection Attacks** - Core prompt manipulation techniques
